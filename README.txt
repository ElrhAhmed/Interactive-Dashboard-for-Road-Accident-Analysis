This project is an application for visualizing road traffic accidents in Morocco. It uses several Python libraries to create interactive charts and maps to represent accident data.


Before running this program, make sure you have installed the following libraries:

pandas
matplotlib
geopandas
tkinter
tkintermapview
PySide6
openpyxl
shapely
You can install these libraries using pip.



To ensure a clear and correct view of the project, adjust the following screen settings:

Open the settings
Go to screen settings:
Set screen resolution to: 1920x1080
Set screen scaling to: 100%


Run the Main.py file.
Click the buttons to switch between different visualizations.
Hover over the visualizations to display the legend (key).
